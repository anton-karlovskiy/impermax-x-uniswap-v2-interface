
# Impermax Finance UI

## Quick setup

1. Take a copy of `.env.sample` and re-name to `.env`
2. Enter the details into the `.env` file

## Running

``` bash
$ yarn install # install dependencies
$ yarn start # serve with hot reload at localhost:3000
$ yarn build # build for production
```
